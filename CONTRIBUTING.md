# Contributing

Thank you for contributing to Hazen Solutions.

## Before Contributing

Please ensure that:

- The proposed change has a clear purpose.
- Existing engineering standards are followed.
- Documentation is updated when necessary.
- Commit messages follow Conventional Commits.

## Pull Requests

Each pull request should focus on a single logical change.

Large pull requests should be split into smaller, reviewable units whenever possible.

## Engineering Principles

All contributions should follow the Engineering Principles documented in the Engineering Handbook.
